# Pustaka Digital

## Project Title

Pustaka Digital

## Project Description

Pustaka Digital is a web-based digital library application developed as an academic project. It allows users to browse books, search for titles, view book information, and manage their favorite books through a simple and responsive interface.

## LIVE WEBSITE LINK
https://adellftrhh.github.io/PustakaDigital/

## GITHUB REPOSITORY LINK
https://github.com/adellftrhh/PustakaDigital

## Features

- User login authentication.
- Informative home page and dashboard.
- Browse digital book catalogue.
- Search books by title, categories or author name.
- Add selected books to borrow into basket or cart.
- Book donation.
- Purchase membership.
- Edit profile and change account settings.
- Library contact information.
- Change language for easy understanding.
- Responsive interface for desktop and mobile devices

## Instructions to Test Login

- Click the website.
- Sign up.
- Fill in the information required and create account.
- Login using the newly created account.
